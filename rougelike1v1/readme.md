# 单骑无双模式简介

此为三国杀OL-单骑无双模式的新月杀移植版，力求忠于原作。

## 玩法

- 模式由2人进行，游戏开始时先决定先后手
- 选择武将后，双方选择1个初始战法（免费）
- 牌堆准备时，使用标+EX+军争牌堆，但是移除全部坐骑牌
- 每个回合结束后，双方获得虎符并进行消费。首个回合结束时，先手获得2符、后手获得3符；
  第1、2、3轮的回合结束后，双方各获得1符；第4轮及以后的轮次中改为双方各获得2符
- 商店中玩家可以选择购买商品、锁定商店（0费，防止下次刷新）、刷新商店（1费）。购买完成后商店自动刷新。
- 每名玩家最多从商店中获得2个额外技能。
